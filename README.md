# Portainer Brute Force
A Simple Python Program that uses Selenium to Brute Force Portainer Web Interface using Firefox Driver
## Requirements
Langauge Used = Python3<br />
Modules/Packages used:
* requests
* datetime
* optparse
* multiprocessing
* colorama
* time
